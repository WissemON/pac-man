- Trello board export: `evidence/trello-pac-man.json`
- Git history summary: `evidence/git-history.txt`

## Generated Documents

- `timeline.md`: planned and actual timeline, with a Mermaid Gantt view.
- `kanban_tracking.md`: Trello Kanban state and card tracking.
- `actual_progress_vs_timeline.md`: progress compared with the planned sequence.
- `git_activity_summary.md`: commit-based evidence and contribution summary.
- `project_analysis_and_choices.md`: technical/project choices and rationale.
- `risk_analysis.md`: risks, mitigations, and status.
- `team_organization.md`: roles, responsibilities, and decision process.
- `acceptance_test_plan.md`: tested features, evidence, and bugs fixed.
- `blocking_points_and_conflicts.md`: blockers, conflicts, and resolution summary.
- `evidence_summary.md`: quick index of proof sources.
